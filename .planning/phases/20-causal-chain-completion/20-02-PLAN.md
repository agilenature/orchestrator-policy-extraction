**session_start.py changes:**

The hook uses stdlib only (http.client, json, socket) -- no heavy imports. For writing the BUS_REGISTRATION_FAILED event to JSONL, do NOT import from src.pipeline.premise.staging (that module uses fcntl which is fine, but we want session_start.py to remain self-contained with minimal imports for hook startup speed). Instead, implement a lightweight JSONL append directly in the file.

1. Add a `_append_event_to_staging(event: dict)` helper function that writes a single JSON line to a staging JSONL file. Use a configurable path via env var or default:
   ```python
   _STAGING_PATH = os.environ.get(
       "OPE_SESSION_STAGING_PATH", "data/session_events_staging.jsonl"
   )

   def _append_event_to_staging(event: dict) -> None:
       """Append a governance event to the session events staging JSONL file.

       Lightweight: no external imports, no file locking (hook runs once per session).
       Creates parent directory if needed.
       """
       try:
           path = pathlib.Path(_STAGING_PATH)
           path.parent.mkdir(parents=True, exist_ok=True)
           with open(path, "a") as f:
               f.write(json.dumps(event, default=str) + "\n")
       except Exception:
           pass  # fail-open: event write failure does not block session
   ```
   Add `import pathlib` at the top (stdlib, no cost).

2. Add environment variable reads for repo, project_dir, and transcript_path:
   ```python
   _OPE_REPO = os.environ.get("OPE_REPO", "")
   _OPE_PROJECT_DIR = os.environ.get("OPE_PROJECT_DIR", "")
   _OPE_TRANSCRIPT_PATH = os.environ.get("OPE_TRANSCRIPT_PATH", "")
   ```

   Note: `OPE_TRANSCRIPT_PATH` is included here so that live sessions can send `transcript_path` to the bus alongside `repo` and `project_dir`. Without this, the `transcript_path` column added by Plan 20-01 would never be populated by any real session. The env var will be set by the governing orchestrator (or manually) when the transcript file path is known at session start time.

3. Modify `main()` to detect whether OPE_RUN_ID was set from environment (i.e., was provided by the governing orchestrator) vs. locally generated:
   ```python
   def main() -> None:
       session_id = _OPE_SESSION_ID or f"session-{datetime.now(timezone.utc).isoformat()}"
       run_id = _OPE_RUN_ID or session_id  # fallback pre-OpenClaw

       # Determine if OpenClaw is providing the run_id
       openclaw_unavailable = not bool(_OPE_RUN_ID)

       # Build register payload with new fields
       register_payload = {
           "session_id": session_id,
           "run_id": run_id,
       }
       if openclaw_unavailable:
           register_payload["openclaw_unavailable"] = True
       if _OPE_REPO:
           register_payload["repo"] = _OPE_REPO
       if _OPE_PROJECT_DIR:
           register_payload["project_dir"] = _OPE_PROJECT_DIR
       if _OPE_TRANSCRIPT_PATH:
           register_payload["transcript_path"] = _OPE_TRANSCRIPT_PATH

       # Register with bus
       result = _post_json("/api/register", register_payload)

       # Detect bus unavailability: _post_json returns {} on connection failure.
       # Empty dict is falsy in Python (bool({}) is False), so `if not result:`
       # correctly detects bus failure.
       if not result:
           _append_event_to_staging({
               "event_type": "BUS_REGISTRATION_FAILED",
               "session_id": session_id,
               "run_id": run_id,
               "attempted_at": datetime.now(timezone.utc).isoformat(),
               "openclaw_unavailable": openclaw_unavailable,
           })

       # Get constraint briefing (rest of main unchanged)
       check = _post_json("/api/check", {
           "session_id": session_id,
           "run_id": run_id,
           "premise_data": {},
       })

       constraints = check.get("constraints", [])
       if constraints:
           count = len(constraints)
           print(f"\n[OPE] {count} active constraint(s) for this session.", flush=True)
           forbidden = [c for c in constraints if c.get("severity") == "forbidden"]
           if forbidden:
               print(
                   f"[OPE] {len(forbidden)} FORBIDDEN constraint(s) in scope:",
                   flush=True,
               )
               for c in forbidden[:3]:
                   print(
                       f"[OPE]   - {c.get('text', c.get('id', ''))[:80]}",
                       flush=True,
                   )
   ```

Key behavioral requirements:
- `_post_json` returns `{}` when bus is unavailable (existing behavior). Empty dict is falsy in Python (`bool({}) is False`), so `if not result:` correctly detects bus failure.
- The BUS_REGISTRATION_FAILED event is written to the staging JSONL, not to stdout. It carries session_id, run_id, attempted_at.
- `openclaw_unavailable` is True when OPE_RUN_ID env var was empty/absent.
- `transcript_path` is included in the register payload when `OPE_TRANSCRIPT_PATH` env var is set.
- The session still proceeds normally after emitting the event (fail-open).
