Create `src/pipeline/doc_indexer.py` implementing the full indexing pipeline.

**Constants:**
- `TIER3_STOPWORDS = frozenset({'not', 'vs', 'as', 'to', 'the', 'a', 'an', 'in', 'of', 'for', 'is'})` -- per locked Pitfall 5
- `DEFAULT_BUS_SOCKET = "/tmp/ope-governance-bus.sock"`

**Functions (all must be independently testable):**

1. `load_axis_vocabulary(db_path: str = "data/ope.db", memory_md_path: str | None = None) -> list[str]`
   - Primary: query `SELECT DISTINCT ccd_axis FROM memory_candidates WHERE LENGTH(TRIM(ccd_axis)) > 0` from DuckDB (read_only=True)
   - Fallback: parse MEMORY.md with regex `r'\*\*CCD axis:\*\*\s+`?([^`\n]+)`?'` (same as nversion.py)
   - Always include 'always-show' in the returned list (it's a doc_index-only concept)
   - Return sorted, deduplicated list
   - Fail gracefully: return empty list on any error

2. `_parse_frontmatter_axes(content: str) -> list[str]`
   - **CRITICAL (Pitfall 4):** Frontmatter MUST start on line 0 with exactly `---`. Files starting with `#` or any other content have NO frontmatter.
   - Find closing `---` (first occurrence after line 0)
   - Parse YAML block using `yaml.safe_load()` (PyYAML is available)
   - Extract `axes` key -- accept both list format (`axes: [a, b]`) and YAML list format
   - Return empty list if no frontmatter or no `axes` key

3. `_axis_in_headers_or_comments(content: str, axis: str) -> bool`
   - Match H1/H2 headers: `r'^#{1,2}\s+.*' + re.escape(axis) + r'.*$'` with MULTILINE | IGNORECASE
   - Match HTML comments: `r'<!--\s*ccd:\s*' + re.escape(axis) + r'\s*-->'` with IGNORECASE
   - Return True if either matches

4. `_axis_token_match(content: str, axis: str, min_tokens: int = 2, min_total: int = 3) -> bool`
   - Split axis on `-`, filter out TIER3_STOPWORDS
   - If fewer than 2 non-stopword tokens remain, return False (axis too short for reliable matching)
   - Count occurrences of each token (case-insensitive) in content
   - Return True only if `len(matched_tokens) >= min_tokens` AND `total_count >= min_total`

5. `extract_axes(content: str, known_axes: list[str]) -> list[dict[str, Any]]`
   - Run 3-tier cascade: Tier 1 (frontmatter, conf=1.0) -> Tier 2 (regex, conf=0.7) -> Tier 3 (keyword, conf=0.4)
   - Each tier only checks axes NOT already found by a higher tier
   - Return list of dicts: `{ccd_axis, association_type, extracted_confidence}`
   - If no tiers fire: return `[{ccd_axis: 'unclassified', association_type: 'unclassified', extracted_confidence: 0.0}]`

6. `_extract_description(content: str, max_length: int = 200) -> str`
   - Skip YAML frontmatter (if line 0 is `---`, skip to line after closing `---`)
   - Skip blank lines, heading lines (starting with `#`), horizontal rules (`---`)
   - Take first non-empty prose paragraph (consecutive non-blank, non-heading lines)
   - Join with spaces, truncate to max_length with `...` suffix if needed
   - Return empty string if no prose found

7. `_content_hash(content: str) -> str`
   - `hashlib.sha256(content.encode('utf-8')).hexdigest()[:16]` -- consistent with project pattern

8. `_bus_is_running(socket_path: str = DEFAULT_BUS_SOCKET) -> bool`
   - Attempt `socket.AF_UNIX` connect to socket_path with 1.0s timeout
   - Return True if connection succeeds, False on any exception (ConnectionRefusedError, FileNotFoundError, OSError)
   - Close socket in finally block

9. `reindex_docs(db_path: str = "data/ope.db", docs_dir: str = "docs", socket_path: str = DEFAULT_BUS_SOCKET, memory_md_path: str | None = None) -> dict[str, int]`
   - Check bus running: if `_bus_is_running(socket_path)`, print error to stderr and raise `SystemExit(1)` with message: `"[ERROR] Bus daemon is running. Stop it first: python -m src.pipeline.cli bus stop"`
   - Load axis vocabulary via `load_axis_vocabulary(db_path, memory_md_path)`
   - Walk `docs_dir` recursively for `*.md` files using `pathlib.Path.rglob('*.md')`
   - For each file: read content, compute hash, extract axes, extract description
   - Open DuckDB connection to db_path (NOT read_only -- this is the write path)
   - Ensure doc_index table exists: `from src.pipeline.live.bus.doc_schema import create_doc_schema; create_doc_schema(conn)`
   - DELETE FROM doc_index (full refresh per locked decision)
   - INSERT all rows: `(doc_path, ccd_axis, association_type, extracted_confidence, description_cache, section_anchor, content_hash)`
     - doc_path: relative to repo root (e.g., `docs/guides/GOVERNING-ORCHESTRATOR-METHODOLOGY.md`)
     - section_anchor: NULL for Phase 21 (file-level only)
   - Print summary to stdout: indexed count, unclassified count, axis distribution
   - Print unclassified docs to stderr for human review (per locked decision)
   - Return summary dict: `{total_files, indexed_rows, unclassified_files, axes_found}`
