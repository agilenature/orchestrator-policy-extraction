**2. Implement `src/pipeline/shadow/evaluator.py`:**

```python
class ShadowEvaluator:
    """Compare a shadow recommendation against the actual human decision."""

    def evaluate(self, episode: dict, recommendation: Recommendation) -> dict:
        """Evaluate a single recommendation against the actual episode decision.

        Returns a result dict matching shadow_mode_results table columns.
        """
```

The evaluate method:
- Extract human_mode, human_risk from episode (mode, risk columns)
- Extract human_reaction_label from episode
- Compare mode: mode_agrees = (recommendation.recommended_mode == human_mode)
- Compare risk: risk_agrees = (recommendation.recommended_risk == human_risk)
- Compute scope_overlap: Jaccard similarity between recommendation.recommended_scope_paths and episode's orchestrator_action.scope.paths (if available). Jaccard = |intersection| / |union|, returning 0.0 if both empty or 1.0 if both empty (no paths to compare = agreement)
- Compare gates: gate_agrees = set(recommendation.recommended_gates) == set(episode gates)
- Collect danger info from recommendation.is_dangerous, recommendation.danger_reasons
- Generate shadow_run_id via uuid4()
- Return dict matching shadow_mode_results columns

**3. Implement `src/pipeline/shadow/runner.py`:**

```python
class ShadowModeRunner:
    """Run shadow mode testing in batch over historical episodes."""

    def __init__(self, conn, embedder, recommender, evaluator=None):
        self._conn = conn
        self._embedder = embedder
        self._recommender = recommender
        self._evaluator = evaluator or ShadowEvaluator()

    def run_all(self, batch_id: str | None = None) -> dict:
        """Run shadow mode for ALL episodes in the database.

        Returns aggregate stats: {total, agreements, dangerous, results_by_session}
        """

    def run_session(self, session_id: str, batch_id: str | None = None) -> list[dict]:
        """Run shadow mode for all episodes in a single session."""
```

For `run_all()`:
1. Fetch all distinct session_ids from episodes table
2. Generate batch_id if not provided (uuid4)
3. For each session, call run_session()
4. Write all results to shadow_mode_results table
5. Return aggregate stats

For `run_session()`:
1. Fetch all episodes for the session from episodes table
2. For each episode:
   a. Extract observation and orchestrator_action
   b. Call recommender.recommend(observation, orchestrator_action, exclude_episode_id=episode_id)
   c. Call evaluator.evaluate(episode, recommendation)
   d. Add batch_id and session_id to result
3. Return list of result dicts

Write results to DuckDB using INSERT OR REPLACE (idempotent for re-runs with same episode_ids).

**1. Implement `src/pipeline/shadow/reporter.py`:**

```python
class ShadowReporter:
    """Compute and format shadow mode metrics from stored results."""

    def __init__(self, conn: duckdb.DuckDBPyConnection):
        self._conn = conn

    def compute_report(self, batch_id: str | None = None) -> dict:
        """Compute aggregate metrics from shadow_mode_results table.

        If batch_id provided, filter to that batch. Otherwise use all results.

        Returns dict with:
        - total_episodes: int
        - total_sessions: int (distinct session_ids)
        - mode_agreement_rate: float (0-1)
        - risk_agreement_rate: float (0-1)
        - avg_scope_overlap: float (0-1)
        - gate_agreement_rate: float | None (0-1, None if no gate data)
        - dangerous_count: int
        - danger_categories: dict[str, int] (count per danger type)
        - meets_threshold: bool (mode_agreement_rate >= 0.70)
        - meets_session_minimum: bool (total_sessions >= 50)
        - per_session: list[dict] with session_id, episode_count, mode_agreement_rate
        """
```

Query shadow_mode_results:
```sql
SELECT
    COUNT(*) as total,
    COUNT(DISTINCT session_id) as sessions,
    AVG(CASE WHEN mode_agrees THEN 1.0 ELSE 0.0 END) as mode_rate,
    AVG(CASE WHEN risk_agrees THEN 1.0 ELSE 0.0 END) as risk_rate,
    AVG(scope_overlap) as avg_scope,
    AVG(CASE WHEN gate_agrees THEN 1.0 ELSE 0.0 END) as gate_rate,
    SUM(CASE WHEN is_dangerous THEN 1 ELSE 0 END) as dangerous
FROM shadow_mode_results
WHERE (? IS NULL OR run_batch_id = ?)
```

For per-session breakdown, GROUP BY session_id.

For danger_categories, parse the danger_reasons JSON column and count each category.
