<objective>
Build foil instantiation (historical episode lookup + divergence detection), staining pipeline (from AmnesiaDetector + derivation propagation), and staging ingestion (JSONL to DuckDB bridge). Integrate all three into the batch pipeline runner.

Purpose: Foil instantiation is the projective layer -- it predicts where the current premise path and the foil path would diverge, using historical evidence. Staining is the retrospective feedback loop -- when OPE Layer 5 finds a constraint violation, premises that relied on the violated constraint are marked stained. Staging ingestion closes the two-writer gap between the PAG hook (Plan 02) and the batch pipeline.

Output: `src/pipeline/premise/foil.py`, `src/pipeline/premise/staining.py`, `src/pipeline/premise/ingestion.py`. Updated `src/pipeline/runner.py`.

    **`src/pipeline/premise/foil.py`**: Foil instantiation with divergence detection.

    `class FoilInstantiator`:
    - `__init__(self, registry: PremiseRegistry, conn: duckdb.DuckDBPyConnection)`: Store registry and connection references.

    - `instantiate(self, foil_text: str, project_scope: str | None = None, current_session_id: str | None = None) -> list[FoilMatch]`:
      Three-tier matching approach (RESEARCH.md Open Question 4):
      1. **Exact claim match**: Call `registry.find_by_foil(foil_text, project_scope, exclude_session=current_session_id)`.
      2. **Keyword overlap** (if tier 1 returns <3 results): Extract significant words (>3 chars, not stopwords) from `foil_text`. Query premise_registry for premises whose `claim` contains 3+ of those keywords. Use a simple SQL WHERE clause with multiple ILIKE conditions joined by AND.
      3. Return up to 10 results total as `FoilMatch` objects.

    - `detect_divergence(self, foil_match: FoilMatch) -> DivergenceNode | None`:
      Given a FoilMatch (which contains the historical premise and its session/episode context):
      1. Look up the episode linked to the foil premise via `ground_truth_pointer.episode_id`.
      2. If episode found, extract `orchestrator_action` JSON from episodes table.
      3. Look up events for that episode session from the events table (limited to events within the episode's time range).
      4. For Phase 14.1 **simplified heuristic**: compare just the `tool_name` sequence. The divergence node is the first event where `event_type` differs between foil episode events and a baseline (the current session's recent events). If no events available for comparison, return None.
      5. Return `DivergenceNode(event_index, foil_tool, claim_tool, episode_id)` or None.

    `class FoilMatch(BaseModel, frozen=True)`:
    - `premise: PremiseRecord` (the historical premise whose claim matches the foil)
    - `match_tier: Literal["exact", "keyword"]`
    - `keyword_overlap: int = 0` (number of overlapping keywords, for tier 2)
    - `divergence_node: DivergenceNode | None = None` (populated by detect_divergence)

    `class DivergenceNode(BaseModel, frozen=True)`:
    - `event_index: int` (position in event sequence where divergence occurs)
    - `foil_tool: str` (tool name in the foil episode at divergence point)
    - `claim_tool: str` (tool name in the current context at divergence point, or "unknown")
    - `episode_id: str` (the foil episode containing the divergence)

    **`src/pipeline/premise/staining.py`**: Staining pipeline.

    `class StainingPipeline`:
    - `__init__(self, registry: PremiseRegistry)`: Store registry reference.

    - `stain_from_amnesia(self, amnesia_events: list[AmnesiaEvent]) -> list[str]`:
      For each AmnesiaEvent:
      1. Get all premises in the registry for the same session_id.
      2. For each premise, check if its `validated_by` text mentions the violated constraint_id (simple substring check: `amnesia_event.constraint_id in premise.validated_by`).
      3. If match found: call `registry.stain(premise_id, stained_by=amnesia_event.amnesia_id, ground_truth_pointer={"amnesia_id": amnesia_event.amnesia_id, "constraint_id": amnesia_event.constraint_id, "session_id": amnesia_event.session_id})`.
      4. Return list of stained premise_ids.

    - `propagate_staining(self) -> list[str]`:
      Stolen Concept detection: when a parent premise is stained, child premises citing it in their `derivation_chain` are also stained.
      1. Get all stained premises: `registry.get_stained()`.
      2. For each stained premise, query all other premises whose `derivation_chain` JSON contains the stained premise_id.
         SQL: `SELECT premise_id FROM premise_registry WHERE json_extract_string(derivation_chain, '$[*].derives_from') LIKE '%' || ? || '%' AND (staining_record IS NULL OR json_extract_string(staining_record, '$.stained') != 'true')`
      3. For each found child premise: stain it with `stained_by = "propagation:" + parent_premise_id`.
      4. Use a visited set to prevent infinite loops (max depth 20).
      5. Return list of newly stained premise_ids.

    - `stain_from_policy_violation(self, constraint_id: str, session_id: str) -> list[str]`:
      When PolicyViolationChecker finds a match:
      1. Get all premises for the session.
      2. Check if any premise's `claim` text contradicts the constraint (simple heuristic: premise claim contains "does not apply" or "not relevant" AND validated_by does not reference the constraint).
      3. If match: stain with `stained_by = "policy_violation:" + constraint_id`.
      4. Return list of stained premise_ids.

    **`src/pipeline/premise/ingestion.py`**: Bridge between PAG hook staging and DuckDB premise_registry.

    `def ingest_staging(registry: PremiseRegistry, staging_path: str = STAGING_PATH) -> dict[str, int]`:
    1. Call `read_staging(staging_path)` to get all staged premise dicts.
    2. If empty, return `{"ingested": 0, "skipped": 0, "errors": 0}`.
    3. For each record:
       - Validate by constructing a `PremiseRecord` (Pydantic validation catches malformed data).
       - Call `registry.register(record)`.
       - Count successes and errors.
    4. On success (all or partial), call `clear_staging(staging_path)`.
    5. Return stats dict: `{"ingested": N, "skipped": M, "errors": E}`.
    6. Log via loguru: total ingested, skipped, errors.

    `def run_staining(registry: PremiseRegistry, amnesia_events: list[AmnesiaEvent]) -> dict[str, int]`:
    1. Create `StainingPipeline(registry)`.
    2. Call `staining.stain_from_amnesia(amnesia_events)` -- returns stained IDs.
    3. Call `staining.propagate_staining()` -- returns propagated IDs.
    4. Return `{"direct_stains": len(stained), "propagated_stains": len(propagated)}`.
