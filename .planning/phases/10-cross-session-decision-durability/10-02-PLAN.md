1. **Create `src/pipeline/durability/scope_extractor.py`:**
   - Function `extract_session_scope(events: list[dict]) -> list[str]`:
     Derives session scope_paths from tool call event payloads. Extracts file paths from:
     - `payload.details.file_path` (Read/Edit/Write tools -- these store the path in details)
     - `payload.common.text` for Bash commands -- use a regex to find file path arguments. Match patterns like `/path/to/file.ext` or `src/pipeline/foo.py` (paths with at least one `/` and a file extension, or known source directories). Use a compiled regex, NOT a naive pattern that matches URLs or random text.
     - Normalize paths: strip leading `./`, resolve relative paths, deduplicate
     - Return sorted unique paths. If empty, return `[]` (repo-wide scope per locked decision 6).
   - Note: Parse payload correctly -- it may come as dict or as JSON string from DuckDB (handle both with json.loads fallback, same pattern as runner.py).

2. **Create `src/pipeline/durability/evaluator.py`:**
   - Pydantic model `ConstraintEvalResult(BaseModel, frozen=True)`:
     - `session_id: str`
     - `constraint_id: str`
     - `eval_state: Literal["HONORED", "VIOLATED"]` (UNKNOWN constraints are excluded entirely, never stored)
     - `evidence: list[dict]` -- list of `{event_id, matched_pattern, payload_excerpt}` (max 500 chars per excerpt per locked decision / DurabilityConfig.evidence_excerpt_max_chars)
     - `scope_matched: bool = True`

   - Class `SessionConstraintEvaluator`:
     - Constructor takes `config: PipelineConfig`
     - Method `evaluate(session_id: str, session_scope_paths: list[str], session_start_time: str, events: list[dict], constraints: list[dict], escalation_violations: dict[str, str] | None = None) -> list[ConstraintEvalResult]`:

       For each constraint:
       a. **Temporal check**: Call `get_status_at_time()` logic -- parse `status_history` to find status at `session_start_time`. Use `datetime.fromisoformat()` for comparison (NOT string comparison). Skip if status != "active" or if no history entry exists before session_start_time.
       b. **Temporal existence**: Skip if `session_start_time < constraint["created_at"]` (constraint didn't exist yet).
       c. **Scope check**: Use `scopes_overlap()` from `src/pipeline/utils.py` with `(session_scope_paths, constraint["scope"]["paths"])`. If no overlap, skip entirely (UNKNOWN -- excluded from results per locked decision 2).
       d. **O_ESC auto-violation**: If `escalation_violations` dict maps this `constraint_id` to a `session_id`, mark as VIOLATED with evidence `[{"event_id": "escalation", "matched_pattern": "O_ESC bypass", "payload_excerpt": "Escalation episode bypassed this constraint"}]`.
       e. **Detection hints scan**: For each hint in `constraint.get("detection_hints", [])`, apply case-insensitive substring containment (same as Phase 9 always-bypass matching) against `json.dumps(event["payload"])` for each event. Pre-compile patterns once per constraint (not per event -- Pitfall in research). If any hint matches any event, result is VIOLATED with evidence listing matched events.
       f. **Default**: If scope overlaps but no detection hints matched and no O_ESC violation, result is HONORED (per locked decision 2).

     - The evaluator DOES NOT write to DuckDB -- it returns results. Writing is handled by the writer functions.
     - Note on `escalation_violations`: This is a dict of `{constraint_id: session_id}` built from episodes table where `mode='ESCALATE' AND escalate_bypassed_constraint_id IS NOT NULL AND session_id = current_session`. The caller (runner Step 14 or CLI) queries DuckDB and passes this in.

     - `amnesia_id: str` -- SHA-256(session_id + constraint_id)[:16] for idempotency
     - `session_id: str`
     - `constraint_id: str`
     - `constraint_type: str | None` -- from constraint's `type` field
     - `severity: str | None` -- from constraint's `severity` field
     - `evidence: list[dict]` -- same evidence from ConstraintEvalResult
     - `detected_at: str` -- ISO8601 timestamp

   - Class `AmnesiaDetector`:
     - Method `detect(eval_results: list[ConstraintEvalResult], constraints: list[dict]) -> list[AmnesiaEvent]`:

1. **Create `src/pipeline/durability/index.py`:**
   - Class `DurabilityIndex`:
     - Constructor takes `conn: duckdb.DuckDBPyConnection` and `min_sessions: int = 3`
     - Method `compute_score(constraint_id: str) -> dict`:
       Runs SQL query against `session_constraint_eval` for the given constraint:
       ```sql
       SELECT
           COUNT(*) as sessions_active,
           SUM(CASE WHEN eval_state = 'HONORED' THEN 1 ELSE 0 END) as sessions_honored,
           SUM(CASE WHEN eval_state = 'VIOLATED' THEN 1 ELSE 0 END) as sessions_violated
       FROM session_constraint_eval
       WHERE constraint_id = ? AND eval_state IN ('HONORED', 'VIOLATED')
       ```
       Returns dict: `{"constraint_id": str, "sessions_active": int, "sessions_honored": int, "sessions_violated": int, "durability_score": float | None, "insufficient_data": bool}`.
       If `sessions_active < min_sessions`: `durability_score = None, insufficient_data = True`.
       Else: `durability_score = sessions_honored / sessions_active, insufficient_data = False`.

     - Method `compute_all_scores() -> list[dict]`:
       Runs the grouped SQL query (GROUP BY constraint_id) and returns a list of score dicts for all constraints that have at least one evaluation.

     - Method `get_amnesia_events(session_id: str | None = None) -> list[dict]`:
       Queries the `amnesia_events` table. If session_id provided, filter by it. Returns list of dicts.

2. **Extend `src/pipeline/storage/writer.py`:**
   Add two new functions:

   - `write_constraint_evals(conn, eval_results: list) -> dict[str, int]`:
     Uses INSERT OR REPLACE for idempotent storage. For each ConstraintEvalResult:
     ```python
     conn.execute("""
         INSERT OR REPLACE INTO session_constraint_eval
         (session_id, constraint_id, eval_state, evidence_json, scope_matched, eval_ts)
         VALUES (?, ?, ?, ?, ?, current_timestamp)
     """, [result.session_id, result.constraint_id, result.eval_state, json.dumps(result.evidence), result.scope_matched])
     ```
     Returns `{"written": N}`.

   - `write_amnesia_events(conn, amnesia_events: list) -> dict[str, int]`:
     Uses INSERT OR REPLACE for idempotent storage. For each AmnesiaEvent:
     ```python
     conn.execute("""
         INSERT OR REPLACE INTO amnesia_events
         (amnesia_id, session_id, constraint_id, constraint_type, severity, evidence_json, detected_at)
         VALUES (?, ?, ?, ?, ?, ?, ?)
     """, [event.amnesia_id, event.session_id, event.constraint_id, event.constraint_type, event.severity, json.dumps(event.evidence), event.detected_at])
     ```
     Returns `{"written": N}`.
