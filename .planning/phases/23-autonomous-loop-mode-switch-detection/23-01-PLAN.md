Create `src/pipeline/ebc/detector.py` with:

```python
class EBCDriftDetector:
    WRITE_TOOLS = frozenset({"Edit", "Write"})
    BASH_WRITE_INDICATORS = frozenset({"mkdir", "cp ", "mv ", "touch ", "> ", ">> ", "tee "})

    def __init__(self, config: PipelineConfig) -> None:
        ebc_cfg = config.ebc_drift
        self._threshold = ebc_cfg.threshold
        self._tolerance_patterns = ebc_cfg.tolerance_patterns
        self._write_tool_names = frozenset(ebc_cfg.write_tool_names)
        self._bash_write_indicators = frozenset(ebc_cfg.bash_write_indicators)

    def detect(
        self,
        ebc: ExternalBehavioralContract,
        session_events: list[dict],
        session_id: str,
    ) -> EBCDriftAlert | None:
```

Detection algorithm:
1. Extract actual file write paths from session_events by scanning for:
   - Events with tool_name in `self._write_tool_names`: extract `file_path` from `details` dict in payload.
   - Events with tool_name == "Bash": check if command text contains any `self._bash_write_indicators`; if so, try to extract file path from command (best-effort, not critical).
   - Events with tool_name in Read/Glob/Grep/WebFetch/WebSearch: SKIP (read-class, not write).
2. Normalize paths: strip leading `./`, make relative to project root.
3. Filter out tolerance patterns: if any file path matches a pattern in `self._tolerance_patterns` (use fnmatch), exclude it from the unexpected set.
4. Compute drift signals:
   - `unexpected_files`: files in actual_writes but NOT in `ebc.expected_write_paths` -> weight 1.0 each.
   - `missing_expected_files`: files in `ebc.expected_write_paths` but NOT in actual_writes -> weight 0.3 each.
5. Compute drift_score: `sum(signal.weight for signal in signals) / max(len(ebc.expected_write_paths), 1)`, capped at 1.0.
6. If drift_score < self._threshold: return None.
7. Otherwise return EBCDriftAlert with all fields populated.

Add private helper `_extract_write_paths(self, session_events: list[dict]) -> set[str]` that does the tool scanning.

Add private helper `_file_weight(self, path: str) -> float` that returns 0.0 for tolerance pattern matches, 0.1 for `__init__.py` or test files, 1.0 for everything else.
