**Modify `src/pipeline/ddf/intelligence_profile.py`**: Add structural integrity to profile computation.

Add a new function `_compute_integrity_for_profile(conn, session_ids, subject) -> tuple[float | None, int]`:
- Query structural_events for the given subject across all provided session_ids.
- Count total structural events.
- If no structural events exist, return (None, 0).
- Compute average integrity score across sessions: for each session_id with structural events, call `compute_structural_integrity()` from structural/computer.py. Average the non-None scores.
- Return (avg_integrity_score, total_structural_event_count).
- Wrap in try/except (ImportError, Exception) so that if structural module is not available, return (None, 0) gracefully.

**Modify `compute_intelligence_profile()`**: After computing spiral_depth (line 171), add:
```python
    # Structural integrity (Phase 18)
    integrity_score = None
    structural_event_count = 0
    try:
        from src.pipeline.ddf.structural.computer import compute_structural_integrity
        # Get sessions for this human
        session_rows = conn.execute(
            "SELECT DISTINCT session_id FROM flame_events "
            "WHERE subject = 'human' AND human_id = ? "
            "AND (assessment_session_id IS NULL)",
            [human_id],
        ).fetchall()
        session_ids = [r[0] for r in session_rows]
        if session_ids:
            scores = []
            total_se = 0
            for sid in session_ids:
                result = compute_structural_integrity(conn, sid, 'human')
                if result is not None:
                    scores.append(result.integrity_score)
                    total_se += result.structural_event_count
            if scores:
                integrity_score = round(sum(scores) / len(scores), 4)
                structural_event_count = total_se
    except (ImportError, Exception):
        pass  # Structural module not available or no data
```
Then pass `integrity_score=integrity_score, structural_event_count=structural_event_count` to the IntelligenceProfile constructor.
