CREATE TABLE IF NOT EXISTS project_wisdom (
    wisdom_id    VARCHAR PRIMARY KEY,
    type         VARCHAR NOT NULL CHECK(type IN ('breakthrough','dead_end','scope_decision','method_decision')),
    title        VARCHAR NOT NULL,

Methods:

1. `insert(entity: WisdomEntity, embedding: list[float]) -> None`:
   - `INSERT OR REPLACE INTO project_wisdom` with all columns
   - Cast embedding as `?::FLOAT[384]`
   - `json.dumps()` for episode_ids and metadata columns
   - Per locked decision: idempotent via INSERT OR REPLACE

2. `bm25_search(query: str, type_filter: str | None = None, limit: int = 20) -> list[tuple[WisdomEntity, float]]`:
   - SQL using `fts_main_project_wisdom.match_bm25(wisdom_id, ?)`
   - WHERE `score IS NOT NULL` and optional `AND type = ?`
   - Confidence filter: `AND confidence_score >= 0.7` only when type_filter='dead_end'
   - ORDER BY score DESC LIMIT ?
   - Parse rows back into WisdomEntity objects (handle JSON columns with json.loads)

3. `vector_search(query_embedding: list[float], type_filter: str | None = None, limit: int = 20) -> list[tuple[WisdomEntity, float]]`:
   - SQL using `array_cosine_similarity(embedding, ?::FLOAT[384]) AS sim`
   - WHERE `embedding IS NOT NULL` and optional type filter
   - Confidence filter: `AND confidence_score >= 0.7` only when type_filter='dead_end'
   - ORDER BY sim DESC LIMIT ?
   - Parse rows back into WisdomEntity objects

4. `rebuild_indexes() -> None`:
   - Install and load fts extension
   - PRAGMA create_fts_index('project_wisdom', 'wisdom_id', 'description', 'content_for_embedding', stemmer='porter', stopwords='english', lower=1, overwrite=1)
   - Install and load vss extension
   - DROP INDEX IF EXISTS wisdom_hnsw
   - CREATE INDEX wisdom_hnsw ON project_wisdom USING HNSW (embedding)
   - CRITICAL: No metric parameter on HNSW index (DuckDB 1.4.4 bug)

5. `count(type_filter: str | None = None) -> int`:
   - SELECT count(*) with optional type filter

6. `list_all(type_filter: str | None = None) -> list[WisdomEntity]`:
   - SELECT * with optional type filter, returns list of WisdomEntity

7. Helper `_row_to_entity(row) -> WisdomEntity`:
   - Converts DuckDB row tuple to WisdomEntity
   - json.loads() for episode_ids and metadata if they are strings
