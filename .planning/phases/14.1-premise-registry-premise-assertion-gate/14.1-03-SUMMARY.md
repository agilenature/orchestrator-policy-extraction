## Decisions Made
- Used CASE WHEN ILIKE positional counting for keyword search rather than full-text search (simpler, sufficient for Phase 14.1)
- Simplified divergence detection to adjacent-tool-change heuristic (first differing tool call in sequence)
- Used CAST(derivation_chain AS VARCHAR) LIKE for JSON array child lookup (DuckDB JSON functions vary by version)
- Runner integration re-reads amnesia events from DuckDB for staining rather than passing in-memory objects (avoids scope coupling)
