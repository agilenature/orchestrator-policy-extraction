**`__init__.py`**: Export StructuralEvent, StructuralConfig, StructuralIntegrityResult.

**`models.py`**: Three frozen Pydantic v2 models:

1. `StructuralEvent(BaseModel, frozen=True)`:
   - Fields: event_id (str), session_id (str), assessment_session_id (Optional[str]=None), prompt_number (int), subject (Literal["human","ai"]), signal_type (Literal["gravity_check","main_cable","dependency_sequencing","spiral_reinforcement"]), structural_role (Optional[str]=None), evidence (Optional[str]=None), signal_passed (bool), score_contribution (Optional[float]=None), contributing_flame_event_ids (list[str]=Field(default_factory=list)), op8_status (Optional[Literal["pass","fail","na"]]=None), op8_correction_candidate_id (Optional[str]=None), created_at (datetime with UTC default)
   - `make_id(session_id, prompt_number, signal_type) -> str`: SHA-256 of f"{session_id}:{prompt_number}:{signal_type}" truncated to [:16]. Follow FlameEvent.make_id pattern exactly.

2. `StructuralConfig(BaseModel)`:
   - Fields: gravity_window (int=3), gravity_weight (float=0.30), main_cable_weight (float=0.40), dependency_weight (float=0.20), spiral_weight (float=0.10), neutral_fallback (float=0.5)
   - This is NOT frozen -- it's a config model (matches OAxsConfig pattern).

3. `StructuralIntegrityResult(BaseModel, frozen=True)`:
   - Fields: session_id (str), subject (str), integrity_score (Optional[float]), gravity_ratio (float), main_cable_ratio (float), dependency_ratio (float), spiral_capped (float), structural_event_count (int=0)

**`schema.py`**: DDL constant + creation function:

- `STRUCTURAL_EVENTS_DDL`: CREATE TABLE IF NOT EXISTS structural_events with all 14 columns from CONTEXT.md locked schema. Include CHECK constraints on subject IN ('human','ai'), signal_type IN ('gravity_check','main_cable','dependency_sequencing','spiral_reinforcement'), op8_status IN ('pass','fail','na') OR op8_status IS NULL. Use VARCHAR[] for contributing_flame_event_ids.
- `STRUCTURAL_EVENTS_INDEXES`: list of 3 index DDL strings (idx_structural_session on session_id, idx_structural_signal on signal_type, idx_structural_subject on subject).
- `create_structural_schema(conn)`: Execute DDL + indexes. Idempotent (IF NOT EXISTS).

**`writer.py`**: Idempotent writer function:

- `write_structural_events(conn, events: list[StructuralEvent]) -> int`: INSERT OR REPLACE INTO structural_events for each event. Pass contributing_flame_event_ids as Python list directly (DuckDB handles VARCHAR[] natively -- do NOT JSON-serialize). Return len(events). Early return 0 if empty list.
