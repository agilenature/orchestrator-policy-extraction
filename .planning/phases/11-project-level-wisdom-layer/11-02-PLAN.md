**src/pipeline/wisdom/retriever.py — WisdomRetriever class:**

Separate class from HybridRetriever (NOT extending it), per locked decision.

```python
class WisdomRetriever:
    def __init__(self, store: WisdomStore, embedder: EpisodeEmbedder, top_k: int = 3):
        self._store = store
        self._embedder = embedder
        self._top_k = top_k
```

Methods:

1. `retrieve(query: str, conn: duckdb.DuckDBPyConnection) -> list[WisdomRef]`:
   - Generate query embedding via `self._embedder.embed_text(query)`
   - Call `self._store.bm25_search(query, limit=self._top_k * 2)` — over-fetch for RRF
   - Call `self._store.vector_search(query_embedding, limit=self._top_k * 2)` — over-fetch
   - Apply RRF fusion (same formula as HybridRetriever: score = sum(1/(60 + rank)))
   - Return top-k as `list[WisdomRef]` with `is_warning=False`

2. `detect_dead_ends(query: str, conn: duckdb.DuckDBPyConnection) -> list[WisdomRef]`:
   - Per locked decision: dual BM25 + vector top-10 agreement
   - Generate query embedding
   - Call `self._store.bm25_search(query, type_filter="dead_end", limit=10)`
   - Call `self._store.vector_search(query_embedding, type_filter="dead_end", limit=10)`
   - Extract wisdom_ids from each result set
   - `agreed_ids = bm25_ids & vss_ids` (set intersection)
   - Filter: confidence_score >= 0.7 (already applied in store search when type_filter="dead_end")
   - Return matching entities as `list[WisdomRef]` with `is_warning=True`
   - If no agreement: return empty list

3. Helper `_entity_to_ref(entity: WisdomEntity, is_warning: bool = False) -> WisdomRef`:
   - Maps WisdomEntity fields to WisdomRef

4. Static method `_rrf_fuse(bm25_results, vss_results, k, rrf_k=60) -> list[str]`:
   - Same RRF formula as HybridRetriever._rrf_fuse but returns wisdom_ids sorted by score
   - Input: list[tuple[WisdomEntity, float]] from each search
   - Output: list of wisdom_id strings in fused rank order, limited to k

Update `src/pipeline/wisdom/__init__.py` to also export WisdomRetriever.
